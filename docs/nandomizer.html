Questionable Modules

Nandomizer

Randomize inputs.

Nandomizer randomly selects a connected input to send to the output every time you trigger the gate.

Fade When the gate is triggered and switches input sources this controls how much the previous input fades out. 0.f has no fade and 1.f is full fadeout effect.

Gate triggers the random swap.

Output outputs the current randomly selected input.

Inputs 1-8 are inputs that take in any monophonic value. These will be randomly selected once you send voltage to the gate input.

Themes

The modules context menu has a theme submenu where you can change the background to a solid white or black color if you are having trouble with the default background design.
